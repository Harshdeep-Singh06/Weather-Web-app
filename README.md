# Weather Check

A simple weather website that shows real-time weather for any city.

## Features

* Search weather by city
* Shows temperature, humidity, and wind speed
* Weather icons change based on condition
* Handles invalid city searches

## Built With

* HTML
* Tailwind CSS
* JavaScript
* OpenWeather API

## How It Works

1. Enter a city name.
2. The app fetches weather data from OpenWeather API.
3. The UI updates with temperature, humidity, wind speed, and weather icon.
